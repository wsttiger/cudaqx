# Chromobius Reference Data

This directory contains small reference inputs for the CUDA-Q QEC Chromobius
decoder tests.

The detector error model in `basic_reference.dem` and the syndrome/prediction
pairs in `basic_reference.tsv` are adapted from the upstream
`quantumlib/chromobius` test case `TEST(main_predict, basic)` in
`src/chromobius/commands/main_predict.test.cc`.
